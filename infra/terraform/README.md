# infra/terraform

Infrastructure as Code (IaC) using Terraform.

## Contents
- Cloud provider setup (AWS/GCP/Azure)
- Compute resources
- Networking
- Databases
- Storage buckets
- CDN configuration
- Variables and outputs

## Modules
- `networking/` - VPC, subnets, security groups
- `compute/` - Kubernetes, VMs
- `database/` - RDS, Postgres
- `storage/` - S3, GCS
- `cdn/` - CloudFront, Akamai
