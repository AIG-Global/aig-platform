# Public Demo Script: "Diana's First Day"

**Date:** 2026-07-06  
**Purpose:** First public demo of North Star ONE (v0.2 Alpha)  
**Duration:** 10 minutes  
**Audience:** Potential users, investors, press  

---

## Demo Overview

This script shows the complete Phase 1 feature set through a single 10-step journey. It demonstrates:

1. User creates account
2. Diana greets user
3. User asks Diana to help with a project
4. Diana generates documents
5. User creates a task list
6. User searches their memory
7. User searches for relevant knowledge
8. User installs a new skill
9. User uses Diana again for document edits
10. User continues work on phone (Beam Me Up)

**Total Time:** 10 minutes  
**Story:** A product manager using Diana to plan a product launch

---

## Act 1: Authentication (1 minute)

### Scene: Signup Screen

**Visual:**
- Browser loads North Star ONE homepage
- Diana avatar in corner, friendly smile
- "Welcome to North Star ONE" headline
- Simple signup form

**Demo Steps:**

1. Click "Get Started" button
2. Enter email: `demo@northstarone.io`
3. Enter password: `DemoDay2026!`
4. Check "I agree to terms"
5. Click "Create Account"

**Expected Result:**
- ✅ Account created
- ✅ Redirected to dashboard
- ✅ Diana greets user: "Welcome! I'm Diana. I'm here to help you accomplish your goals."

**Talking Points:**
- "Fast signup - no friction"
- "Diana immediately establishes relationship"
- "We've removed the complexity of product onboarding"

## Act 2: First Conversation (2 minutes)

### Scene: Diana Chat Interface

**Visual:**
- Diana in chat sidebar
- Chat area with "Start a conversation with Diana"
- Text input at bottom

**Demo Steps:**

1. Click chat input box
2. Type: "I need help planning our product launch for next month"
3. Press Enter / Click Send

**Diana's Response (Streaming):**

```
I'd love to help you plan this launch! Let me break this down...

To create a successful launch, we typically need:

1. **Timeline & Milestones** - What's your target launch date?
2. **Audience & Positioning** - Who are we launching to?
3. **Marketing & PR** - What's the go-to-market strategy?
4. **Internal Alignment** - Who needs to be involved?
5. **Success Metrics** - How will we measure launch success?

Why don't you tell me about each of these, and I'll help you create 
a comprehensive launch plan?

(Response streams in real-time)
```

**Expected Result:**
- ✅ Streaming response appears character-by-character
- ✅ Response is contextually relevant
- ✅ Diana shows expertise and asks follow-up questions

**Talking Points:**
- "Diana isn't just a chatbot - she's an expert advisor"
- "Real-time streaming makes it feel natural and immediate"
- "Each response is tailored to your needs"
