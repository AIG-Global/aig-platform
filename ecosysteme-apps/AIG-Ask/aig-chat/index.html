Connect your Anthropic API key

AIG Ask talks directly to Claude's API from your browser. There's no backend here — your messages go straight to Anthropic and back, and your key is stored only in this browser's local storage. This app and its author never see either.

  • 1
    Create an Anthropic account Go to console.anthropic.com and sign up if you haven't already.
  • 2
    Add billing The API is pay-as-you-go (not the same as a claude.ai subscription). You'll need to add a payment method in the console under "Billing" — typical chat usage costs a few cents per conversation.
  • 3
    Create an API key In the console, go to Settings → API Keys and create a new key. It starts with sk-ant-.
  • 4
    Paste it below Then you're chatting.
Your key is saved with localStorage in this browser only. It's sent on every message directly to api.anthropic.com — never to any other server. Clearing your browser data or using a different browser/device means setting it again.
