## Asset Types

### Type 1: Applications
Complete applications that extend AIGINVEST.
- Example: CRM, Task Manager, Project Tracker
- Revenue: 30% to developer, 70% to AIGINVEST
- Approval: Requires security review, functionality test
- Versions: Multi-version support, 6-month sunset period

### Type 2: Skills
Pre-built Diana capabilities that extend her abilities.
- Example: Web Research, Email Draft, Data Analysis
- Revenue: 50% to creator, 50% to AIGINVEST
- Approval: Automatic if tested
- Installation: One-click for Diana integration

### Type 3: Mission Packs
Pre-built mission templates with tasks and guidance.
- Example: "Launch a Startup", "Learn Coding", "Build a Team"
- Revenue: 25% to creator, 75% to AIGINVEST
- Content: 20+ structured tasks
- Diana Integration: Mission-specific guidance included

### Type 4: Templates
Documents, documents, and workflows.
- Example: Business Plan, Contract, Proposal
- Revenue: 20% to creator, 80% to AIGINVEST
- Formats: Editable, versionable
- AI Completion: AI can auto-fill sections

### Type 5: Themes (Future)
Design themes and visual packs.
- Customization: Colors, fonts, layouts
- Apply to: Web, Mobile, AIOS

### Type 6: Agents (Future)
Specialist AI agents beyond Diana.
- Example: Research Agent, Coding Agent, Finance Agent
- Coordination: Via Diana orchestrator

## Publishing Process

### Developer Submission
1. Create listing in developer console
2. Upload asset package
3. Provide documentation
4. Submit for review

### Review Gates
- **Security:** No malware, no phishing, no privacy violations
- **Performance:** <3s load, <100MB storage
- **Functionality:** Works as advertised
- **Compliance:** Follows guidelines, no illegal content

### Approval
- Automatic for skills (if tests pass)
- 48-hour for apps
- Instant for templates
- Manual for agents

### Launch
- Listed in marketplace
- Available for discovery
- Start earning revenue

## Revenue Sharing

### Payment Model
- Monthly payouts
- Minimum threshold: $50
- Preferred: Direct to bank account
- Alternative: AIGINVEST wallet

### Commission Structure
```
Apps:           30% developer, 70% AIGINVEST
Skills:         50% creator, 50% AIGINVEST
Mission Packs:  25% creator, 75% AIGINVEST
Templates:      20% creator, 80% AIGINVEST
```

### Growth Incentives
- **Top performer bonus:** Extra 5% if >$1K MRR
- **Quality bonus:** Extra 5% if >4.5 rating
- **Ecosystem growth:** Share 10% of platform revenue

## Compliance & Certification

### Developer Certification
- Level 1: Individual developer (can sell)
- Level 2: Verified seller (faster review)
- Level 3: Premium partner (featured)

### Content Guidelines
- No malware or exploits
- No harassment or hate speech
- No illegal content
- No privacy violations
- No competing with core AIGINVEST

### Dispute Resolution
- User can request refund
- Developer can respond
- Mediation by AIGINVEST if needed
- Escalation to legal if required
