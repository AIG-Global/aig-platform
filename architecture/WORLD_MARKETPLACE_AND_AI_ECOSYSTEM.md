## Executive Summary

**World Marketplace (WDM)** is the central commerce and services hub within AIOS, positioning AIG as the premium business operating system for corporations, governments, and high-net-worth individuals.

Unlike consumer marketplaces (Amazon, eBay), WDM focuses on **business-critical goods and services**:
- Luxury goods & lifestyle services
- Professional B2B services & consulting
- Enterprise AI software & automation
- Investment & wealth management
- Travel & hospitality premium services
- Real estate & property
- Vehicles & transportation
- Strategic partner integrations

WDM is **integrated with AI Apps bundled into membership tiers**, creating unprecedented value stacking:

```
MEMBERSHIP BUNDLES:
├─ Pack 2 (€699/month) = 3 AI apps
├─ Pack 3 (€1,099/month) = 5 AI apps + Market Alerts
└─ Pro Pack (€2,999/month) = All AI apps + Six Degrees of Separation + future AI
```

This transforms AIOS from a network platform into a **comprehensive business operating system** where members receive increasingly powerful AI assistance as they progress.

  app               AIApp @relation(fields: [appId], references: [id], onDelete: Cascade)
  
  tierBundle        String   // "pack_2", "pack_3", "pro" - which bundle includes it
  
  active            Boolean  @default(true)
  subscriptionDate  DateTime @default(now())
  
  @@unique([userId, appId])
}

// Update Membership to include app bundles
model Membership {
  id                String   @id @default(cuid())
  userId            String   @unique
  user              User @relation(fields: [userId], references: [id], onDelete: Cascade)
  
  organizationId    String?
  organization      Organization? @relation(fields: [organizationId], references: [id])
  
  // Original tier system
  tier              String   // "free", "starter", "professional", "business", "platinum", "vendor"
  
  // NEW: Bundle system
  bundle            String?  // "pack_2", "pack_3", "pro" - takes precedence if set
  
  bundleStartDate   DateTime?
  bundleRenewalDate DateTime?
  
  commissionRate    Int      @default(10) // 10-70%
  
  active            Boolean  @default(true)
  
  apps              AIAppSubscription[]
  
  createdAt         DateTime @default(now())
  updatedAt         DateTime @updatedAt
}
