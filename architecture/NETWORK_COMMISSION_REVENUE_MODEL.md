## EXECUTIVE SUMMARY

The AIGINVEST ecosystem operates on a **network-driven revenue model** where:
- Members choose their own **network commission percentage (10-70%)**
- Commission split: **80% Network Bonuses + 20% Management Pool**
- This applies uniformly across all revenue streams
- **Whole network earns when members enjoy, invest, and buy**

### Commission Split Logic

```
Network Commission (10-70%) Split:
├─ 80% → Network Bonuses
│  └─ Distributed to upline members (MLM structure)
│  └─ Based on commission depth & tier
│  └─ Follows unilevel structure (6-10 levels)
│
└─ 20% → Management Pool
   └─ AIGINVEST operations & platform maintenance
   └─ Used for: customer support, infrastructure, R&D, marketing
   └─ Transparent accounting available to members
```

### Application to All Revenue Streams

**Same 80/20 commission split applies to:**

1. **WDM Products** (Member-selected commission)
   - Seller lists product at €100
   - Seller chooses 10-70% network commission
   - 80% of commission → Network bonuses
   - 20% of commission → Management pool

2. **Investment Packages** (Fixed structure)
   - Member invests €1,000 in fund
   - Network commission: Automatic (varies by fund)
   - 80% of commission → Network bonuses
   - 20% of commission → Management pool

3. **Outside Service Offerings** (Provider-set commission)
   - Service provider (designer, consultant, etc.)
   - Sets commission percentage (10-70%)
   - 80% of commission → Network bonuses
   - 20% of commission → Management pool

4. **Membership Fees** (LOCKED at 80/20)
   - **€399 Starter tier**
     - Network bonuses: €319.20 (80%)
     - Management pool: €79.80 (20%)
   - **€699 Professional tier**

     - Management pool: €139.80 (20%)

## PART 5: COMMISSION CALCULATION LOGIC

### For WDM Products

```typescript
function calculateWDMProductCommission(
  productPrice: number,
  networkCommissionPercent: number // 10-70
) {
  // Validate commission range
  if (networkCommissionPercent < 10 || networkCommissionPercent > 70) {
    throw new Error('Network commission must be between 10-70%')
  }

  const networkCommission = productPrice * (networkCommissionPercent / 100)
  const networkBonus = networkCommission * 0.80 // 80% to network
  const managementPool = networkCommission * 0.20 // 20% to management
  const sellerReceives = productPrice - networkCommission

  return {
    productPrice,
    networkCommissionPercent,
    networkCommissionAmount: networkCommission,
    networkBonusAmount: networkBonus,
    managementPoolAmount: managementPool,
    sellerReceives,
  }
}

// Example: €100 product with 30% commission
const result = calculateWDMProductCommission(100, 30)
// {
//   productPrice: 100,
//   networkCommissionPercent: 30,
//   networkCommissionAmount: 30,
//   networkBonusAmount: 24,
//   managementPoolAmount: 6,
//   sellerReceives: 70
// }
```

### For Membership Tiers

```typescript
function calculateMembershipCommission(tierPrice: number) {
  const networkBonus = tierPrice * 0.80 // 80% of tier price
  const managementPool = tierPrice * 0.20 // 20% of tier price

  return {
    tierPrice,
    networkBonusAmount: networkBonus,
    managementPoolAmount: managementPool,
    distributionByLevel: {
      level1: networkBonus * 0.30, // 30%
      level2: networkBonus * 0.20, // 20%
      level3: networkBonus * 0.15, // 15%
      level4: networkBonus * 0.10, // 10%
      level5: networkBonus * 0.03, // 3%
      level6: networkBonus * 0.02, // 2%
      // (Remaining levels 7-10 get 0 for Starter, but would get 0 anyway)
    },
  }
}

// Example: €399 Starter tier

//     level4: 31.92,   // 10%
```

### For Investment Packages

```typescript
function calculateInvestmentCommission(
  investmentAmount: number,
  investmentType: string, // 'tag_markets', 'crypto', 'funds', 'aigphone', 'aigio'
) {
  // Investment commission varies by type
  const commissionPercentByType = {
    tag_markets: 15,
    crypto: 10,
    funds: 12,
    aigphone_equity: 20,
    aigio_tokens: 18,
  }

  const commissionPercent = commissionPercentByType[investmentType] || 15
  const networkCommission = investmentAmount * (commissionPercent / 100)
  const networkBonus = networkCommission * 0.80
  const managementPool = networkCommission * 0.20

  return {
    investmentAmount,
    investmentType,
    commissionPercent,
    networkCommissionAmount: networkCommission,
    networkBonusAmount: networkBonus,
    managementPoolAmount: managementPool,
    memberInvestment: investmentAmount, // Full amount invested
  }
}
```

### Example: Month 1 Transaction Flow

```
Scenario: New member joins + makes first purchase + recruits downline

Member A (Starter tier: €399/month, 6-level depth):
├─ Joins platform
│  └─ Network bonus distributed: €319.20 → Network uplines
│  └─ Management pool: €79.80
│
├─ Lists product (€100 with 30% commission)
│  └─ Product Commission (€100 sale):
│     ├─ Network commission: €30
│     ├─ Network bonuses (€24): → Member A's uplines
│     └─ Management pool (€6)
│  └─ Member A seller revenue: €70
│
└─ Recruits Member B
   └─ Member B joins Starter (€399)
      └─ Network bonus distributed: €319.20 → Includes Member A (level 1)
      └─ Member A receives: €319.20 × 30% = €95.76
      └─ Management pool: €79.80

Total Month 1 Earnings for Member A:
├─ Own membership commission: €319.20 (80% of own €399 fee)
├─ Product sales commission: €24 (80% of 30% on €100 product)
├─ Downline commission (Member B): €95.76 (30% × €319.20)
├─ Seller revenue from product: €70
└─ TOTAL: €509.16 (profit margin: €510.16 after €399 fee = €110.16)
```

## PART 7: MEMBER INCENTIVE STRUCTURE

### Why Members Choose Higher Commission Percentages

**Lower Commission (10%):**
- Attracts price-conscious buyers
- Higher seller margins
- Slower network growth

**Higher Commission (70%):**
- Attracts network-minded sellers
- Lower seller margins but higher commission volume
- Accelerates network growth
- Better for scaling

**Optimal Strategy:**
```
New sellers: 20-30% commission (balance buyer appeal + network growth)
Established sellers: 40-50% commission (brand recognition + network benefit)
Premium sellers: 50-70% commission (premium positioning)
```
