## The User Journey (What We're Building)

### Meet Diana: 7 Steps

**Step 1: Visit AIGINVEST**
- Homepage answers three questions
- Beautiful, inviting, clear CTA

**Step 2: Create Account**
- Email/password registration
- 90 seconds, smooth, polished
- Works on mobile

**Step 3: Meet Diana**
- Diana greets user personally
- Explains capabilities clearly
- Three action buttons

**Step 4: Start Conversation**
- User types message
- Diana responds (streaming)
- Conversation feels natural

**Step 5: Create Project**
- User asks Diana for something concrete
- Diana creates value (project structure)
- User sees it immediately

**Step 6: Persistence**
- Everything is saved to database
- No data loss
- All work is safe

**Step 7: Return Tomorrow**
- User logs in next day
- Everything is restored
- Diana remembers context
- User continues where they left off
