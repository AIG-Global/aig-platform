## If Anything Doesn't Work

### If the Agent Gets Blocked

**They'll tell you at 5 PM or earlier.**

Ask: "What do you need from me?"

Examples:
- "I need database credentials" → Provide them
- "I need a design decision" → Make it
- "I need a scope decision" → Decide

**Then:** "Keep building. I'll remove this blocker."

### If Something Takes Longer

**Timeline adjusts, quality doesn't.**

At standup the next day:
"Yesterday took longer than planned. Today we finish it."

**Never:** Skip testing, skip docs, ship broken code.

### If You Don't Like the Direction

**Say so at the demo.**

"I want to modify this approach. Here's what I'm thinking..."

**Then:** Make the decision, communicate clearly, move forward.
