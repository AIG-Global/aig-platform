## The 15-Minute Alpha Test

**Success:** New user installs Diana and within 15 minutes:
1. ✅ Meets Diana (feels welcomed)
2. ✅ Asks something meaningful
3. ✅ Gets helpful response with context

5. ✅ Feels Diana genuinely understands them

**Outcome:** "Diana is my new colleague" (not "cool app")

## Decision Framework (Use This Daily)

**When faced with any decision:**

1. Does it align with Diana being a trusted companion?
2. Does it improve Diana's core capability?
3. Can we explain it in one sentence?
4. Would users notice if we shipped it?
5. Would we regret not shipping it?

**5 yes → Do it now**
**4 yes → Schedule it**
**3 yes → Consider it**
**< 3 yes → Skip it**
