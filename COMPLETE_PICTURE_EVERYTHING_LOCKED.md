## If Something Goes Wrong

### Problem Solving Process

1. **Identify the issue** — What's not working?
2. **Find the root cause** — Why?
3. **Find the fastest fix** — What's the solution?
4. **Ask for help if needed** — Unblock immediately
5. **Move forward** — Don't let it stop the day

### Example
**Track B:** "Docker won't run on Windows PowerShell."  
**Founder:** "Can you use WSL or a native PostgreSQL instead?"  
**Track B:** "Yes, doing that now."  
**Founder:** "Good. Keep going."
