# @aig/auth

Authentication and authorization library for the AIG platform.

## Purpose
Shared authentication layer providing:
- Identity verification
- Session management
- Permission checking
- Role-based access control (RBAC)
- OAuth/OIDC integration
- Token management

## Usage
```typescript
import { AuthService, authorize } from '@aig/auth';
```

## Features
- [ ] JWT authentication
- [ ] Session tokens
- [ ] Permission guards
- [ ] OAuth providers
- [ ] MFA support
