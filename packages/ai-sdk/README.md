# @aig/ai-sdk

AI/ML SDK and utilities for AI-powered features across the platform.

## Purpose
Shared AI integration library providing:
- LLM model interfaces
- RAG (Retrieval-Augmented Generation) utilities
- Prompt engineering helpers
- Vector database integration
- AI model orchestration

## Usage
```typescript
import { AIClient } from '@aig/ai-sdk';
```

- [ ] LLM Client abstraction
- [ ] Vector embeddings
