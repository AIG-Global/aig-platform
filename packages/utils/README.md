# @aig/utils

Common utility functions and helpers used across the platform.

## Purpose
Shared utility library providing:
- Data transformers
- Validation helpers
- Date/time utilities
- String utilities
- Array utilities
- Math helpers

## Usage
```typescript
import { formatDate, validate } from '@aig/utils';
```
