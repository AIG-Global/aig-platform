# @aig/logging

Structured logging library for the platform.

## Purpose
Shared logging utilities providing:
- Structured logging
- Log levels and filtering
- Log aggregation
- Performance monitoring
- Error tracking

## Usage
```typescript
import { Logger } from '@aig/logging';
```

## Features
- [ ] Winston/Pino integration
- [ ] Structured logs
- [ ] Log levels
- [ ] Error context
- [ ] Performance metrics
