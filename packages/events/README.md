# @aig/events

Event bus and event management library for platform-wide communication.

## Purpose
Shared event system providing:
- Pub/sub messaging
- Event types and schemas
- Event handlers
- Event persistence
- Event sourcing utilities

## Usage
```typescript
import { EventBus, emit } from '@aig/events';
```

## Features
- [ ] Event bus implementation
- [ ] Event schemas
- [ ] Handlers registration
- [ ] Event queuing
- [ ] Event history
