# @aig/notifications

Notification service library for multi-channel messaging.

## Purpose
Shared notification system providing:
- Email notifications
- SMS notifications
- Push notifications
- In-app notifications
- Notification templates
- Delivery tracking

## Usage
```typescript
import { NotificationService } from '@aig/notifications';
```

- [ ] Email provider
- [ ] SMS provider
- [ ] Push notifications
- [ ] Notification templates
- [ ] Delivery queue
