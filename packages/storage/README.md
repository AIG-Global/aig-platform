# @aig/storage

Storage abstraction layer for file and object storage.

## Purpose
Shared storage utilities providing:
- S3/cloud storage abstraction
- File upload/download
- CDN integration
- File validation
- Storage optimization

## Usage
```typescript
import { StorageClient } from '@aig/storage';
```

## Features
- [ ] S3 adapter
- [ ] Azure Blob adapter
- [ ] GCS adapter
- [ ] File upload
- [ ] Pre-signed URLs
- [ ] Virus scanning
