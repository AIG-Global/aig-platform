Public documentation website and developer portal.

Public-facing documentation site for:
- API documentation
- Developer guides
- Getting started tutorials
- SDK documentation
- Integration examples
- Best practices
- FAQ and troubleshooting

- Next.js 14 for frontend
- MDX for documentation content
- Nextra or Docusaurus alternative
