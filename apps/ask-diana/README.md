# apps/ask-diana

Ask Diana - AI-powered assistant and conversational interface.

## Purpose
Interactive AI assistant providing:
- Natural language question answering
- Data analysis assistance
- Code generation and debugging
- Content recommendations
- Personalized insights
- Multi-turn conversations
- Integration with platform data

## Technology Stack
- Next.js 14 for frontend (web UI)
- TypeScript
- OpenAI/Claude API integration
- WebSocket for real-time streaming
- Vector database for RAG (Retrieval-Augmented Generation)
- @aig/ai-sdk for AI orchestration

## Features
- [ ] Chat interface
