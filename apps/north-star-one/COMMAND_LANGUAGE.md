# North Star ONE - Command Language System
## Voice Commands & Navigation Terminology

**Status:** Design Reference  
**Device:** North Star ONE (AIOS OS + Diana AI)  
**Last Updated:** 2026-07-07

---

## DESIGN PHILOSOPHY

North Star ONE uses a distinctive **command language** inspired by exploration, astronomy, and guidance. This creates a unique user experience while avoiding IP conflicts with existing franchises.

**Core Principles:**
- Commands feel like **navigating a journey**
- Language emphasizes **guidance and discovery**
- Consistent metaphor: Device guides you like the North Star guides travelers
- Accessible to users of all technical levels
- Memorable and intuitive

```

├─ "Navigate Home"          → Return to main dashboard
├─ "Find My Course"         → Show current status/direction
```

## 15. DESIGN GUIDELINES

### Creating New Commands:

**✅ DO:**
- Use **exploration/astronomy metaphors** (Navigate, Chart, Beacon, Waypoint, Course)
- Keep commands **2-4 words max**
- Make commands **intuitive** for non-technical users
- Provide **context-aware** alternatives
- Test **across different accents/speeds**

**❌ DON'T:**
- Use brand/franchise references (Star Trek, sci-fi franchises)
- Create commands that **conflict with existing** app terminology
- Use technical jargon without plain-language alternative
- Overload with synonyms (pick one primary command)
- Use commands that are **hard to pronounce**

### Examples of Good Command Naming:

```
GOOD ✅                    WHY
Navigate Home              Clear, directional
Chart Route                Metaphor-consistent
Send to Beacon             Poetic but understandable
Create Waypoint            Exploration-themed
Diana, Stand By            Natural, respectful
Sync Course                Implies synchronization

AVOID ❌                   WHY
Engage                     Too franchise-specific
Beam Me Up                 Direct IP reference
Execute Protocol           Too technical
Initiate Sequence          Jargon-heavy
Lock And Load              Not exploration-themed
```
